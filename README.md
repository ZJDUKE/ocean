# 海南省海洋矿产管理系统

海南省海洋厅海洋矿产资源管理信息化系统，提供矿产勘查监督管理、储量管理、矿业权管理及综合可视化展示。

## 系统模块

| 文件 | 说明 |
|------|------|
| [mineral-dashboard.html](mineral-dashboard.html) | 矿产管理工作台主页 |
| [mineral-exploration.html](mineral-exploration.html) | 勘查开发日常监督管理 |
| [mineral-reserves.html](mineral-reserves.html) | 矿产储量管理 |
| [mineral-rights.html](mineral-rights.html) | 矿业权管理 |
| [mineral-one-map.html](mineral-one-map.html) | 海洋矿产一张图场景 |
| [mineral-detail.html](mineral-detail.html) | 矿产疑似位置详情页（含证据链与企业） |

## 技术栈

- HTML5 + CSS3 + JavaScript
- Chart.js 图表库
- 二三维 GIS 可视化接口
- 深海洋主题设计系统
